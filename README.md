# Centralized Judiciary System

## Overview

The Centralized Judiciary System is a web-based application designed to manage legal cases, users, and administrative processes in a centralized platform.

## My Contribution

* Designed frontend UI using HTML, CSS, and Bootstrap
* Developed admin and user dashboards
* Built structured forms for case management
* Created responsive layout and navigation

## Technologies Used

* HTML
* CSS
* JavaScript
* Java (JSP / Servlets / Spring)
* MySQL

## Features

* Admin dashboard for managing users and cases
* User interface for accessing case details
* Case management system
* Form-based data entry
* Structured and responsive UI
